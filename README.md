# Community Service Request and Response Management System with ML-Based Priority Ranking

This is a hybrid AI system that integrates Machine Learning for priority prediction and Knowledge Representation & Reasoning for rule-based refinements.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Train the model: `python src/ml_model.py`
3. Run the UI: `streamlit run src/app.py`

## Project Structure

- `data/`: Dataset
- `models/`: Trained model and encoders
- `src/`: Source code
- `notebooks/`: Jupyter notebooks for analysis
- `documentation/`: Docs
- `video/`: Demo videos
